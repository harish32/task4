Api Documentation

/meta

https://loaclhost:5000/meta?url=yoururl&tag=tagname

example:

http://localhost:5000/meta?url=https://twitter.com&tag=description

output

{success: true,data: "From breaking news and entertainment to sports and politics, get the full story with all the live commentary."}
by default the tagname is looked in name attribute if you want to change the attribute you can pass as attr
example:

<meta property="og:image" content="https://www.facebook.com/images/fb_icon_325x325.png" />

http://localhost:5000/meta?url=https://facebook.com&tag=og:image&attr=property

output

{success: true,data: "https://www.facebook.com/images/fb_icon_325x325.png"}
